# Prebid Js Project Set up

## PreRequisities
1. Node JS
2. VS Code
3. Git

1. Git clone the forked repository
```bash
git clone https://github.com/Rise-Media-Technologies/risemedia-prebid-js.git
```

2. Navigate to the repository and install dependencies
```bash
npm install
```

3. Build the project
(Using Gulp)

```bash
gulp serve
```
